# Screenshot Guide

One entry per screenshot. Capture each in sequence during the lab.

---

## Task 1: Launch the Lightsail instance

### 1. `screenshot-lightsail-console-home.png`

**Console action:** Open the Lightsail console home page

**What must be visible:** The Lightsail dashboard with the "Create instance" button clearly visible in the top area of the page

**Trainer note:** Capture the full dashboard width so the button is unambiguous

---

### 2. `screenshot-select-blueprint.png`

**Console action:** Select the OpenClaw blueprint from the application list

**What must be visible:** The blueprint grid with the OpenClaw tile visibly selected/highlighted

---

### 3. `screenshot-instance-plan-selected.png`

**Console action:** Confirm the instance plan and launch

**What must be visible:** The selected plan's price and specs, with the Create instance button visible at the bottom

---

### 4. `screenshot-instance-running.png`

**Console action:** Verify the instance has finished launching

**What must be visible:** The instance row with a green "Running" status badge clearly visible

---

## Task 2: Configure the Gateway and connect a channel

### 5. `screenshot-gateway-access-token.png`

**Console action:** Complete onboarding to generate the access token

**What must be visible:** The onboarding success screen showing a generated access token field

**Trainer note:** Blur or redact the actual token characters before sharing this screenshot publicly

---

### 6. `screenshot-channel-connected.png`

**Console action:** Verify the channel connection succeeded

**What must be visible:** The channel list with the new channel's status clearly showing "Connected"

---

## Task 3: Verify the agent responds

### 7. `screenshot-test-message-response.png`

**Console action:** Send a message and confirm a reply

**What must be visible:** Both the outgoing test message and the agent's incoming reply in the same chat thread

---

## Task 4: Clean up

### 8. `screenshot-instance-deleted.png`

**Console action:** Delete the instance and confirm it no longer appears

**What must be visible:** The Lightsail instance list without the deleted test instance

---
